# PT Career Project Context

## 1. 서비스 정의

PT Career는 운동·재활 전문가의 경력, 자격, 교육 이력, 근무기관, 전문 분야를 한 곳에 정리하고, 소비자가 신뢰할 수 있는 전문가를 찾을 수 있도록 돕는 전문가 프로필 플랫폼이다.

초기 MVP는 예약, 결제, 후기, 채팅 기능을 제공하지 않는다. 소비자는 전문가 프로필을 확인한 뒤 센터 주소, 전화번호, 홈페이지, 외부 문의 링크를 통해 직접 센터에 연락하거나 방문한다.

## 2. Mission

PT Career는 운동·재활 전문가의 경력과 자격을 투명하게 보여주어, 소비자가 자신의 상황에 맞는 전문가를 신뢰하고 선택할 수 있도록 돕는 플랫폼이다.

## 3. Core Philosophy

좋은 전문가는 원래 입소문이 난다.

PT Career는 그 입소문이 오프라인에서 끝나지 않고, 카카오톡·문자·DM·링크 공유를 통해 온라인에서도 신뢰와 함께 이어질 수 있도록 만든다.

## 4. 핵심 차별점

PT Career는 단순 검색 사이트가 아니라 “온라인 명함 + 신뢰 전파 플랫폼”이다.

모든 전문가는 공유 가능한 고유 프로필 URL을 가진다. 누군가 좋은 전문가를 소개할 때 PT Career 프로필 하나만 보내면, 받은 사람은 사진, 경력, 자격, 전문 분야, 근무기관, 위치 정보를 빠르게 확인할 수 있다.

## 5. 주요 사용자

### 일반 소비자
- 통증, 재활, 운동, 체형관리, 스포츠 복귀 등을 목적으로 전문가를 찾는 사람
- 회원가입 없이 전문가 목록과 프로필을 열람할 수 있어야 한다.

### 전문가
- 물리치료사
- 퍼스널 트레이너
- 건강운동관리사
- 선수트레이너
- 필라테스 강사
- 재활운동 전문가

## 6. MVP 범위

포함:
- 로그인
- 전문가 프로필 등록
- 경력·자격·교육 이력 관리
- 지도 기반 전문가 탐색
- 전체 전문가 목록
- 전문가 상세 프로필
- 온라인 명함 링크
- 프로필 공유
- 관리자 자격 검토

제외:
- 예약
- 결제
- 후기
- 채팅
- AI 추천
- 커뮤니티
- 교육 판매
- 채용
- 앱 전용 기능

## 7. North Star Metric

공유된 전문가 프로필 수

공유는 단순 조회보다 강한 신뢰의 신호다. 누군가 프로필을 공유했다는 것은 해당 전문가를 추천하거나 확인할 가치가 있다고 판단했다는 뜻이다.
