# PT Career PRD v1.0

## 1. Product Vision

PT Career는 운동·재활 전문가의 경력과 자격을 투명하게 보여주어, 소비자가 자신의 상황에 맞는 전문가를 신뢰하고 선택할 수 있도록 돕는 플랫폼이다.

PT Career의 본질은 검색보다 신뢰다. 좋은 전문가의 프로필이 카카오톡, 문자, DM, 링크로 자연스럽게 전달되며 오프라인 입소문이 온라인에서도 이어지게 한다.

## 2. Problem

### 소비자 문제
- 어떤 전문가가 정말 신뢰할 만한지 알기 어렵다.
- 전문가의 경력, 자격, 교육 이력을 확인하기 어렵다.
- SNS나 광고성 정보만으로는 판단하기 어렵다.
- 지인에게 소개받아도 추가 정보를 확인할 곳이 없다.

### 전문가 문제
- 자신의 전문성을 체계적으로 보여줄 공간이 부족하다.
- 경력, 자격, 교육 이력이 흩어져 있다.
- 좋은 입소문이 나도 전달할 수 있는 온라인 프로필이 없다.
- 명함은 있지만 모바일에서 공유 가능한 신뢰 프로필이 없다.

## 3. Product Goal

사용자가 전문가를 검색하고, 프로필을 확인하고, 센터 정보를 통해 직접 문의하거나 방문하도록 만든다.

MVP는 온라인 예약·결제·상담을 제공하지 않는다.

## 4. Core Features

### 4.1 전문가 프로필
- 사진
- 이름 또는 활동명
- 직군
- 전문 분야
- 총 경력
- 한 줄 소개
- 상세 소개
- 근무기관
- 지역
- 면허·자격
- 교육 이력
- 외부 문의 정보

### 4.2 전문가 검색
- 지역 검색
- 전체 전문가 목록
- 전문 분야 필터

### 4.3 온라인 명함
- 모든 전문가는 고유 URL을 가진다.
- 예: `/experts/1`, `/p/joonseok`
- 프로필은 모바일에서 명함처럼 빠르게 읽혀야 한다.

### 4.4 공유 기능
- 링크 복사
- 카카오톡 공유
- 문자 공유
- SNS/DM 공유 대응
- Open Graph 미리보기 지원

공유 미리보기 예시:

도수치료 · 스포츠재활 · 경력 7년

## 5. MVP 포함/제외 범위

### 포함
- 로그인/회원가입
- 전문가 프로필 등록·수정
- 프로필 공개·비공개
- 전체 전문가 목록
- 전문가 상세 페이지
- 지역/분야 검색
- 프로필 공유
- 관리자 자격 검토

### 제외
- 지도 기반 탐색 (Later: M5+)
- 직군 필터 (Later: M4+)
- 경력 필터 (Later: M4+)
- 예약
- 결제
- 리뷰/별점
- 채팅
- AI 추천
- 커뮤니티
- 교육 판매
- 채용
- 모바일 앱

## 6. Success Metrics

### MVP 출시 기준
- 전문가 가입 50명
- 공개 프로필 30개
- 프로필 공유 100회
- 공유 링크 방문 300회
- 센터 문의 클릭 100회

### 핵심 지표
- 프로필 공유 수
- 전문가 프로필 작성 완료율
- 전문가 상세 페이지 조회 수
- 전화/홈페이지/지도 클릭 수
- 공유 링크 유입 수

## 7. UX Principles

- 소비자는 3초 안에 “전문가 찾기”를 이해해야 한다.
- 전문가는 5분 안에 기본 프로필을 만들 수 있어야 한다.
- 프로필은 모바일에서 온라인 명함처럼 보여야 한다.
- 소비자에게 어려운 전문용어만 노출하지 않는다.
- 불필요한 회원가입 장벽을 만들지 않는다.

## 8. Product Philosophy

좋은 전문가는 원래 입소문이 난다.  
PT Career는 그 입소문이 온라인에서도 신뢰와 함께 이어질 수 있도록 만든다.
