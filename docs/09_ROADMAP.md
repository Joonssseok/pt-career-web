# PT Career Roadmap

## Phase 0. Project Foundation

목표:
- 프로젝트 문서 세트 구축
- PRD 확정
- 제품 원칙 확정
- Claude Code 작업 규칙 확정
- 디자인 기준 정리

산출물:
- Project Context
- PRD
- Product Principles
- Brand Guide
- Design System
- Database Design
- Claude Rules
- Decision Log
- Backlog

## Phase 1. MVP Web

목표:
전문가가 프로필을 만들고, 소비자가 검색·지도·공유를 통해 전문가를 찾을 수 있는 웹서비스 출시

기능:
- 로그인
- 프로필 등록/수정
- 전체 전문가 목록
- 전문가 상세 페이지
- 지도 기반 탐색
- 지역/분야 필터
- 프로필 공유
- 관리자 검토

출시 기준:
- 공개 전문가 프로필 30개 이상
- 모바일 사용성 확보
- 링크 공유 미리보기 적용
- 자격 증빙 비공개 저장
- 기본 이용약관/개인정보처리방침 반영

## Phase 2. Growth

목표:
프로필 공급자 확보와 공유 기반 성장 검증

기능:
- 프로필 공유 통계
- QR 명함
- 전문가용 프로필 완성도
- SEO 최적화
- 지역/전문분야 랜딩페이지

KPI:
- 전문가 가입 100명
- 공개 프로필 70개
- 누적 공유 500회

## Phase 3. Community / Education 검토

조건:
- 전문가 프로필 300개 이상
- 공유와 검색 트래픽이 안정적으로 발생
- 전문가들이 재방문할 이유가 필요해질 때

검토 기능:
- 전문가 커뮤니티
- 교육 정보
- 세미나 홍보
- 전문가 콘텐츠

## Phase 4. Hiring / B2B

조건:
- 충분한 전문가 DB 확보
- 병원/센터의 채용 니즈 확인

검토 기능:
- 채용 공고
- 센터 페이지
- 전문가 채용용 프로필

## Phase 5. Reservation / App

조건:
- 법적 리스크 검토
- 실제 소비자 수요 확인
- 운영 인력과 CS 체계 확보

검토 기능:
- 예약
- 앱
- 알림
- 결제
