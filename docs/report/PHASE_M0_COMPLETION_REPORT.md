### 3️⃣ 공개 승인 조건 (14_MVP_SCOPE_V1.md §4.1)

**7가지 필수 조건:**

```
1. 필수 기본 정보 입력
2. 현재 근무기관 1개 이상
3. 대표 전문분야 1개 이상
4. 경력 1개 이상
5. 자격 또는 면허 1개 이상 제출
6. 최소 1개 자격 또는 면허 검증 완료
7. 관리자 최종 승인
```

**정책:**
- 7가지를 모두 만족해야만 공개 가능
- 검토 중인 자격은 일반 사용자에게 미표시
- 본인과 관리자만 검토 상태 확인 가능

### 5️⃣ 자동 알림 범위 (14_MVP_SCOPE_V1.md + 15_MVP_IMPLEMENTATION_PLAN.md)

**MVP에서 제외:**
- 관리자 승인 이메일
- 관리자 반려 이메일
- 푸시 알림

**MVP에 포함:**
- 마이페이지 상태 표시:
  - 작성 중
  - 검토 대기
  - 승인
  - 반려 (+ 관리자 메모)

**M1에 포함:**
- 회원가입 확인 메일 (Supabase Auth)
- 비밀번호 재설정 메일 (Supabase Auth)

**P1로 이동:**
- 승인/반려 자동 이메일
- 푸시 알림

3️⃣ 관리자 검수 (관리자)
