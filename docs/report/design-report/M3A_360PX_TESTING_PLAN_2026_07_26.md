## 2. 각 화면별 테스트 항목

### EXP-002: 프로필 기본정보

#### 레이아웃
- [ ] 모든 입력 필드가 한 컬럼으로 스택됨
- [ ] 버튼 너비 100% (전체 너비 사용)
- [ ] 패딩/마진 적절 (px-4 = 1rem)
- [ ] 텍스트 잘림 없음

#### 입력 필드
- [ ] 이름 입력: maxLength 100 정상 작동
- [ ] 직군 선택: 드롭다운 터치 가능 (높이 44px 이상)
- [ ] 한 줄 소개: maxLength 150 적용
- [ ] 상세 소개: 5줄 텍스트에어리어, maxLength 1000
- [ ] 문자 카운터: X/max 형식 정상 표시

#### 상태 메시지
- [ ] 로딩 상태: "⏳ 저장 중입니다..." 표시
- [ ] 저장 완료: "✓ 저장되었습니다!" 표시
- [ ] 오류 상태: "⚠️ 입력 오류..." + 개별 필드 오류 표시

#### 네비게이션
- [ ] "이전" 버튼: 최소 높이 44px
- [ ] "다음" 버튼: 최소 높이 44px, 전체 너비 flex-1
- [ ] 버튼 간 간격 (gap-3): 적절함
