## 13개 화면 상세 정의

### EXP-ONB-001: 로그인 후 최초 진입

**화면 ID**: EXP-ONB-001

**화면명**: 로그인 후 최초 진입

**사용자**: Google OAuth로 로그인한 신규 사용자 (프로필 없음)

**접근 조건**: `isAuthenticated === true` AND `profile === null`

**화면 목적**: 로그인한 사용자가 아직 전문가 프로필을 만들지 않았음을 이해하고, 프로필 작성 프로세스를 간단히 안내한다.

**진입 경로**: 
- Google OAuth 로그인 성공 후 자동 진입
- 소비자 홈 → 마이페이지 접근 시

**완료 조건**: 사용자가 "전문가 프로필 작성 시작" CTA를 클릭하여 EXP-ONB-002로 이동

**핵심 CTA**: `전문가 프로필 작성 시작` (Primary)

**보조 CTA**: `나중에 하기` (뒤로 가기)

**표시 데이터**:
- 환영 인사말
- 프로필 작성의 목적 (1줄 요약)
- 작성 단계 개요
- 자격 증빙이 필요한 이유
- 승인 전 비공개 안내
- 관리자 검토 후 공개됨 설명

**입력 데이터**: 해당 없음

**필수 항목**: 해당 없음

**선택 항목**: 해당 없음

**읽기 전용 항목**: 해당 없음

**사용자 변경 금지 항목**: 해당 없음

**상태별 UI**:
- **001 (로딩)**: 프로필 조회 중 Skeleton 표시
- **008 (세션 만료)**: "세션이 만료되었습니다. 다시 로그인해주세요."
- **009 (네트워크 오류)**: 프로필 조회 실패 → 재시도 CTA 제공

**정상 라우팅** (상태 매트릭스 제외):
- 기존 프로필이 있는 경우 (profile !== null): 마이페이지 대시보드로 자동 이동
- 이는 권한 거부(007)가 아니라 정상 라우팅으로, 사용자가 이미 프로필을 작성했을 때의 기본 흐름

**오류 처리**: 
- 프로필 조회 실패: 재시도 및 홈으로 이동 옵션

**빈 상태**: 해당 없음 (고정 안내 화면)

**모바일 기준**:
- 360px에서 텍스트 줄바꿈 자연스러움
- CTA 최소 44px 높이
- 상단·하단 여백 충분

**접근성 기준**:
- 모든 텍스트 명도 충분 (대비 4.5:1 이상)
- 포커스 상태 명시
- 의미있는 색상 외에도 텍스트 구분

**연결 화면**: 
- → EXP-ONB-002 (프로필 기본정보)
- ← / (소비자 홈)
- ← /mypage (마이페이지)

**관련 테이블**: `profiles` (읽기 전용)

**관련 RLS 조건**: 
- 사용자가 자신의 프로필만 조회 가능

**Figma 상태**: Draft

**구현 상태**: Not Started

**CTO 결정 필요 사항**: 해당 없음

### EXP-ONB-002: 프로필 기본정보

**화면 ID**: EXP-ONB-002

**화면명**: 프로필 기본정보

**사용자**: 프로필 작성 중인 전문가

**접근 조건**: `isAuthenticated === true` (신규 또는 기존 프로필)

**화면 목적**: 전문가의 기본 정보(이름, 사진, 직군, 한 줄 소개, 상세 소개)를 입력받고 저장한다.

**진입 경로**:
- ← EXP-ONB-001에서 "전문가 프로필 작성 시작" 클릭
- ← EXP-ONB-009에서 편집 링크 클릭
- ← /mypage/profile/edit 직접 접근

**완료 조건**: 모든 필수 정보 입력 + "다음" 클릭 + 저장 성공 → EXP-ONB-003 이동

**핵심 CTA**: `다음` (Primary, 저장 후 이동)

**보조 CTA**: 
- `이전` (이탈 경고 010 포함)
- `프로필 사진 변경` (Secondary)

**표시 데이터**:
- 읽기 전용 필드 (존재 시):
  - 승인 상태 배지
  - 공개 상태 표시
  - 승인일

**입력 데이터**:

**이름 또는 활동명** (필수):
- 유형: 텍스트
- 최소 글자 수: 2자
- 최대 글자 수: 40자
- 타인 사칭, 자격 오인 또는 부적절한 표현: 관리자 검토 대상
- CEO 승인: 최소 2자, 최대 40자

**프로필 사진**:
- 유형: 이미지 파일
- 저장소: Pending Mapping (현재 `profile-images` 후보)
  - M2.1에서 확인: TM-01 프로필 기본정보 매핑에 참조 필드 확인
  - 기술 구현: M2.1에서 현행 구조 확인 및 CTO 권고안 작성, 실제 변경은 CEO 승인 후 진행
- 지원 형식: JPEG, PNG, WebP
- 최대 용량: 5MB
- 필수 여부:
  - 프로필 작성 중: 선택 (사진 없이 임시 저장 가능)
  - 검토 요청 시: 필수 (없으면 미완성 항목으로 안내)
  - 승인 후 공개 프로필: 필수
- CEO 승인: 작성 중 선택, 검토 요청 시 필수

**직군** (필수):
- 유형: 선택 (드롭다운)
- 옵션: MVP 8개 확정 (CEO 승인)
  1. 물리치료사
  2. 퍼스널트레이너
  3. 건강운동관리사
  4. 선수트레이너
  5. 필라테스 지도자
  6. 요가 지도자
  7. 스포츠 코치
  8. 기타 운동·재활 전문가

**직군 선택 원칙**:
- 직군 선택은 면허나 자격 인증을 의미하지 않는다.
- 면허와 자격은 자격·면허 정보 및 증빙자료를 통해 별도로 검토한다.
- 직군 목록은 향후 CEO 승인에 따라 수정될 수 있다.
- CEO 승인: 8개 직군 확정
