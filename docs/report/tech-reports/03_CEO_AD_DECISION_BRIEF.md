# CEO AD Decision Brief — 5개 추가 결정 사항

**발신**: PT Career 디자인팀  
**대상**: CEO  
**작성일**: 2026-07-21

---

## AD-01: 자격번호 필수 여부

**디자인팀 권고**: **선택**

**제품정책**: 자격번호는 선택 입력 (필수: 자격명, 발급기관, 취득일)

**근거**: 온보딩 진입장벽 감소, 번호 없는 자격 대응

---

## AD-02: 증빙파일 허용 형식

**디자인팀 권고**: **PDF, JPEG, PNG**

**제품정책**: 지원 형식만 업로드 가능

**근거**: 모바일 기본 형식, 실행파일 자동 차단

---

## AD-03: 증빙파일 최대 용량

**디자인팀 권고**: **파일당 5MB, 자격당 1개**

**제품정책**: 5MB 초과 시 오류

**근거**: 모바일 업로드 부담과 문서 판독 가능성을 고려한 MVP 제한

---

## AD-04: 센터명·홈페이지 공개범위

**디자인팀 권고**:
```
센터명: 전문가 공개 선택 + 관리자 승인 후 공개

홈페이지: 독립적 공개 선택 또는 센터명 공개와 함께 노출
         (CEO 결정)

상세 주소: MVP 공개 프로필 미노출
```

**근거**: 신뢰도 + 개인정보 분리

**CTO 리스크 검토**: 필요 (RLS 복잡도)

---

## AD-05: 지역 입력 단위

**디자인팀 권고**:
```
선택지 A: 시·도 필수 / 시·군·구 필수
선택지 B: 시·도 필수 / 시·군·구 선택

권장:
- 거주지역: A안 (필수)
- 근무지역: CEO 선택
```

**근거**: 검색 정밀도와 사용성 균형

**CTO 리스크 검토**: 필요 (성능, 유지보수)
