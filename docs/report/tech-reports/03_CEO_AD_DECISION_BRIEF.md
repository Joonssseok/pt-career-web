## M3-A 직접 필수 결정 (3개)

### AD-04: 센터명·홈페이지 공개범위

**현재 승인된 정책**:
- 근무지역: 선택 입력
- 근무지역 공개: 전문가 선택
- 거주지역: 본인 전용
- 개인 연락처: 항상 비공개

**결정할 질문**:
센터명, 홈페이지, 상세 주소의 공개범위는?

**디자인팀 권고**:
- 센터명: 전문가 공개 선택 + 관리자 승인 후 공개
- 홈페이지: 전문가 공개 선택 + 관리자 승인 후 공개
- 상세 주소: MVP 공개 프로필 미노출

**CTO 리스크 검토**: 필요 (RLS 복잡도)

**CEO 결정**:
```
[ ] 권고안 채택
[ ] 수정안: ( )
[ ] 보류
```

### AD-05A: 거주지역 입력 단위

**현재 승인된 정책**:
- 거주지역: 선택 입력
- 거주지역: 본인 전용
- 공개 및 조회 금지

**결정할 질문**:
사용자가 거주지역을 입력할 경우, 시·도까지만 받을지 시·군·구까지 받을지?

**디자인팀 권고**:
거주지역 전체 입력: 선택  
입력 시: 시·도 + 시·군·구 권장

**CTO 리스크 검토**: 필요 (성능, 유지보수)

**CEO 결정**:
```
[ ] 시·도만
[ ] 시·도 + 시·군·구
[ ] 거주지역 입력 제외
```

### AD-05B: 근무지역 입력 단위

**현재 승인된 정책**:
- 근무지역: 선택 입력
- 근무지역: 전문가가 공개 여부 선택

**결정할 질문**:
근무지역을 입력하는 경우, 시·도 + 시·군·구를 모두 필수로 할지, 시·군·구를 선택으로 둘지?

**디자인팀 권고**:
근무지역: 선택 입력  
입력 시: 시·도 + 시·군·구 필수 권장

**CTO 리스크 검토**: 필요 (검색 정밀도)

**CEO 결정**:
```
[ ] 시·도 + 시·군·구 필수
[ ] 시·도 필수, 시·군·구 선택
[ ] 시·도만
```
