# CEO AD Decision Brief

**발신**: PT Career 디자인팀  
**대상**: CEO  
**작성일**: 2026-07-21  
**형식**: 각 AD별 1페이지 / 제품정책만 포함

---

## AD-01: 자격번호 필수 여부

**디자인팀 권고**: **선택**

**제품정책**:
```
자격번호는 선택 입력
- 필수 정보: 자격명, 발급기관, 취득일
- 선택 정보: 자격번호
```

**근거**: 
- 온보딩 진입장벽 감소
- 번호 없는 자격 대응 (민간·국제)
- 검증은 관리팀 증빙파일 검토로 보완

---

## AD-02: 증빙파일 허용 형식

**디자인팀 권고**: **PDF, JPEG, PNG**

**제품정책**:
```
지원 형식: PDF, JPEG, PNG만
거부되는 형식: 기타 모두
사용자 안내: "PDF, JPG, PNG만 지원합니다"
```

**근거**: 
- 모바일 카메라 기본 형식
- 실행파일·압축파일 자동 차단

---

## AD-03: 증빙파일 최대 용량

**디자인팀 권고**: **파일당 5MB, 자격당 1개**

**제품정책**:
```
파일 용량: 5MB 이하
자격당 파일: 최대 1개
초과 시: "파일이 5MB를 초과합니다" 오류
```

**근거**:
- A4 스캔본 충분한 품질
- 일반 네트워크 5초 이내 업로드

---

## AD-04: 센터명·홈페이지 공개범위

**디자인팀 권고**: **선택적 공개**

**제품정책**:
```
센터명: 전문가 공개 선택 시만 노출
홈페이지: 센터명 공개 시 함께 노출
근무기관 상세주소: 공개 프로필에 미노출
근무지역: 기존 정책 (사용자 선택)
```

**근거**:
- 신뢰도 (센터명으로 신원 확인)
- 개인정보 (정확한 주소 후속 단계로)

**CTO 리스크 검토**: 필요  
(공개 여부 저장 필드 + RLS 복잡도)

---

## AD-05: 지역 입력 단위와 옵션 체계

**디자인팀 권고**: **2단계 행정구역**

**제품정책**:
```
1단계 (필수): 시·도 17개
2단계 (선택): 시·군·구 ~250개

적용: 거주지역, 근무지역 모두
```

**근거**:
- 250개 드롭다운 불가 → 2단계 필수
- 표준 행정구역 사용 (임의 선별 제외)
- 지역 필터링 가능

**CTO 리스크 검토**: 필요  
(마스터 데이터 규모 + 드롭다운 성능)
