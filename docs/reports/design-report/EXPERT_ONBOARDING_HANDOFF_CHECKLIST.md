### EXP-ONB-002: 프로필 기본정보

**필수 입력**
- [ ] 이름 또는 활동명 (2~40자, CEO 승인)
- [ ] 직군 (8개 옵션, CEO 승인)

**선택 입력**
- [ ] 프로필 사진 (Pending Mapping — TM-01)
  - 지원 형식: JPEG, PNG, WebP
  - 최대 용량: 5MB
  - 저장소: profile-images (Pending Mapping)
- [ ] 한 줄 소개 (60자 이내, CEO 승인)
- [ ] 상세 소개 (500자 이내, 의료광고 기준 안내)

**상태 요구사항**
- [ ] 필수 정보 누락 (오류 표시)
- [ ] 저장 중 (로딩)
- [ ] 저장 성공 (자동 이동)
- [ ] 저장 실패 (재시도)
- [ ] 파일 업로드 실패 (형식/크기 오류 메시지)
- [ ] 권한 거부
- [ ] 세션 만료
- [ ] 네트워크 오류
- [ ] 이탈 경고 (미저장 변경)
- [ ] 모바일 키보드 (입력 필드 노출)

**기술 요구사항**
- [ ] 테이블: profiles
- [ ] 필드: name, occupation_id, avatar_url, bio_short, bio_long
- [ ] 상태: draft (저장 가능)
