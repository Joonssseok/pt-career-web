## 3. 주요 변경사항

### 3.1 EXP-ONB-002 (프로필 기본정보)

**직군 8개 확정**:
```
1. 물리치료사
2. 퍼스널트레이너
3. 건강운동관리사
4. 선수트레이너
5. 필라테스 지도자
6. 요가 지도자
7. 스포츠 코치
8. 기타 운동·재활 전문가
```

**필수 항목 변경**:
- 필수: 이름, 직군 (2개만)
- 선택: 사진 (검토 요청 시 필수), 한 줄 소개, 상세 소개

### 3.2 EXP-ONB-003 (현재 근무기관) — 중요 변경

**신규 필드 추가**:
- **거주지역** (필수, 비공개): 전문가만 조회·수정
- **근무지역** (필수, 선택적 공개): 전문가가 공개 여부 결정

**주소 정책**:
- MVP: 수동 입력만
- Post-MVP: 검색, 자동완성, 정규화 검토 예정

### 3.3 데이터 공개 범위 (4가지)

| 분류 | 데이터 | 접근 권한 | 공개 범위 |
|------|--------|---------|---------|
| 공개 프로필 | 승인된 기본정보, 경력, 전문분야, 공식 연락처, 근무지역 (선택 시) | 누구나 | 공개 프로필 표시 |
| 소유자 제한 | 개인 연락처, 자격번호, 증빙파일, **거주지역** | 해당 전문가 + 관리자 | 미노출 |
| 선택적 공개 | **근무지역** | 해당 전문가 + 관리자 | 전문가 선택 시 표시 |
| 당사자 제한 | 반려 사유 (user_rejection_reason) | 해당 전문가 + 관리자 | 미노출 |
| 관리자 전용 | 내부 관리자 메모 (internal_admin_notes) | 관리자만 | 절대 미노출 |
