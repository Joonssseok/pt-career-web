## 9번: 거주지역·근무지역 및 공개정책 ⭐ 최우선

### 거주지역

```
입력: 전문가 선택
공개 여부: 절대 비공개
조회 가능: 전문가 본인만
소비자: 조회 불가
다른 전문가: 조회 불가
공개 프로필: 노출 금지
검색 결과: 사용 금지
```

### 근무지역

```
입력: 전문가 선택
공개 여부: 전문가가 공개/비공개 선택
공개 선택 시: 공개 프로필 및 검색 결과에 표시
비공개 선택 시: 소비자에게 미노출
```

### 데이터 공개 범위 (6가지 분류)

| 분류 | 대표 데이터 | 접근 권한 |
|------|-----------|---------|
| **공개 프로필** | 승인된 공개 기본정보, 공개 경력·자격·교육·전문분야, 공식 연락처 | 누구나 |
| **선택적 공개** | 근무지역 | 전문가가 공개를 선택한 경우 소비자 노출 |
| **소유자 전용** | 거주지역 | 해당 전문가 본인만 |
| **소유자·관리자 제한** | 개인 연락처, 자격번호, 증빙자료 | 해당 전문가와 권한 관리자 |
| **당사자 제한** | 사용자 전달용 반려 사유 | 해당 전문가와 권한 관리자 |
| **관리자 전용** | 내부 관리자 메모 | 권한 관리자만 |

**주의**: 거주지역은 관리자도 일반 조회할 수 없음. 실제 RLS 구현방식은 M2 검증 후 별도 승인.
