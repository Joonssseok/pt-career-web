## 🔒 RLS·권한 영향

### 권한 정책 정의

**사용자 (인증된 사용자)**:

| 작업 | 대상 | 권한 |
|------|------|------|
| SELECT | 자신의 프로필·경력·자격 | ✅ 가능 |
| SELECT | 승인된 다른 사용자 프로필 | ✅ 가능 |
| INSERT | 자신의 정보 | ✅ 가능 |
| UPDATE | 자신의 정보 (단, 일부 필드 제외) | ⚠️ 조건부 |
| DELETE | 자신의 정보 | ✅ 가능 |
| UPDATE | verification_status, is_public, approved_at | ❌ 불가능 |

**관리자**:

| 작업 | 대상 | 권한 |
|------|------|------|
| SELECT | 모든 사용자 정보 | ✅ 가능 |
| UPDATE | 프로필 승인 상태 | ✅ 가능 |
| UPDATE | 자격 검증 상태 | ✅ 가능 |
| DELETE | 프로필 | ✅ 가능 |

### RLS 정책 핵심

**사용자 변경 금지 필드** (RLS 강제):
```sql
-- UPDATE 불가능한 필드
verification_status
is_public
approved_at
```

**Storage 권한**:
- `profile-images` (private): 본인만 쓰기, 관리자 읽기
- `evidence-files` (private): 본인만 쓰기, 관리자 읽기
- 공개 URL 발급 금지

### 영향 범위

| 정책 | 영향 화면 | 구현 난이도 |
|------|---------|-----------|
| 사용자 변경 금지 필드 | EXP-ONB-002, 009, 013 | 낮음 (DB 레벨) |
| 관리자만 승인 | EXP-ONB-010, 011, 012, 013 | 낮음 (API 레벨) |
| private Storage | EXP-ONB-002, 006 | 중간 (파일 업로드) |
| signed URL 제한 | 관리자 대시보드 | 중간 (별도 구현) |

## 🔴 CTO 결정 필요 항목

### 7개 항목 (모두 Screen Spec에 명시됨)

#### 1. EXP-ONB-002: 이름 글자 수 제한
- **위치**: 프로필 기본정보 > 이름 입력 필드
- **영향**: DB VARCHAR 길이, 화면 표시 너비
- **옵션**: 예) 50자, 100자, 제한 없음
- **우선순위**: 높음

#### 2. EXP-ONB-002: 프로필 사진 최대 파일 크기
- **위치**: 프로필 기본정보 > 사진 업로드
- **영향**: Storage 할당량, 업로드 시간
- **옵션**: 예) 5MB, 10MB, 20MB
- **우선순위**: 높음

#### 3. EXP-ONB-002: 직군 옵션 목록
- **위치**: 프로필 기본정보 > 직군 선택
- **영향**: 드롭다운 옵션, 마스터 데이터
- **옵션**: 확정된 직군 목록 필요 (예: 물리치료사, 운동처방사, ...)
- **우선순위**: 높음

#### 4. EXP-ONB-002: 한 줄 소개·상세 소개 글자 수 제한
- **위치**: 프로필 기본정보 > 소개 입력
- **영향**: DB VARCHAR 길이, 화면 표시
- **옵션**: 예) 한 줄 소개 150자, 상세 소개 1000자
- **우선순위**: 높음

#### 5. EXP-ONB-003: 지역 옵션 목록
- **위치**: 현재 근무기관 > 지역 선택
- **영향**: 필터링, 소비자 탐색 기준
- **옵션**: 확정된 지역 목록 필요 (예: 서울, 경기, ...)
- **우선순위**: 높음

#### 6. EXP-ONB-003: 주소 API 선택
- **위치**: 현재 근무기관 > 주소 입력
- **영향**: 외부 API 선택 (Kakao/Naver/Google), 비용, 응답 속도
- **옵션**: Kakao 지도 API, Naver 지도 API, Google Places API, 또는 수동 입력만
- **우선순위**: 중간

#### 7. EXP-ONB-004: 최소 필수 경력 개수
- **위치**: 경력 관리 > 검증 로직
- **영향**: 사용자 입력 요구 사항, 프로필 완성도
- **옵션**: 예) 0개(선택), 1개(필수), 2개(필수)
- **우선순위**: 중간
