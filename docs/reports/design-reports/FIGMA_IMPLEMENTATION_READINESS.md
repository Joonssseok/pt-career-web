### 1.4 공개 범위 정책 정리

6-tier 데이터 분류:

| 분류 | 조건 | 공개 범위 |
|------|------|---------|
| 공개 프로필 | 승인 후 | 전체 노출 (로그인 불필요) |
| 선택적 공개 | 사용자 선택 | 공개 선택 시만 노출 |
| 소유자 전용 | 본인만 | 자신의 프로필에서만 조회 |
| 소유자·관리자 제한 | 검토팀 필요 | 소유자 + 관리팀만 |
| 당사자 제한 | 개인정보 | 자신과 관리팀만 |
| 관리자 전용 | 내부 데이터 | 관리팀만 조회·편집 |

**적용 예시**:
- 거주지역: 소유자 전용
- 근무지역: 선택적 공개
- 개인 연락처: 당사자 제한
- 검토 메모: 관리자 전용

### 1.5 모바일 UX 정책

- ✅ 기준: 375px (iPhone SE / 8 기준)
